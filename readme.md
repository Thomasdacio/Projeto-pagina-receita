# Desafio Frontend Mentor - Página de Receita 

Esta é uma solução para o [Desafio da Página de Receita no Frontend Mentor](https://www.frontendmentor.io/challenges/recipe-page-KiTsR8QQKm) 

<img src="./design/gif-pagina-receita.gif" alt="Gif da pagina do projeto pagina de receita">

## Tecnologias utilizadas

- HTML
- CSS

## Coisas que aprendi
Nesse projeto me deparei com o desafio de organizar uma página de tamanho considerável em relação ao que já tinha feito até então. Tive que planejar bem o que faria, como separaria cada seção da minha página e como as estilizaria. Aprendi a usar funções que não tinha usado ainda, como por exemplo:
```
border-collapse: collapse;
```
Que foi utilizada para unir as bordas que foram colocadas na tabela.

## Dificuldades que tive

Minha maior dificuldade foi conseguir posicionar os elementos do jeito que foi proposto pelo desafio. Tive que testar bastante até conseguir um resultado satisfatório
